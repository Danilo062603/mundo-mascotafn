🐾 Bienvenidos a Mundo Mascota

Somos tu plataforma web para el cuidado de mascotas. Encuentra citas veterinarias, tienda online con productos de calidad y mucho más.

🚀 Funcionalidades

Sistema de citas veterinarias, carrito de compras con pedidos en línea y catálogo completo de productos para perros y gatos.

💻 Tecnologías

Desarrollado con HTML, CSS, JavaScript en el frontend y Node.js + Express en el backend para una API REST completa.

🛒 Cómo hacer un pedido

Elige tus productos en la sección de Productos, agrégalos al carrito y finaliza la compra ingresando tus datos de entrega. Recibirás confirmación por pantalla.
